# OFC LitElement TypeScript starter

本项目使用 LitElement + TS 创建：
    
    LitElement 是基于 Web Component 的一个框架，能够简单创建 Web Component 组件。

    实现了跨技术栈（React/Vue）的通用组件。

## Setup

安装：

```bash
npm i
```

## Build

打包生成最新的文件，发布前使用：

```bash
npm run docs
```

监听文件变化，本地启动前需要打开：

```bash
npm run build:watch
```

## Dev Server

本地启动：

```bash
npm run serve
```

本地启动docs：

```bash
npm run docs:serve
```
